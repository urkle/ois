Object Oriented Input System - SDL Fork

This is a small fork of OIS that includes updated support for SDL as an input source.

Changes made in this fork include

  1. Fix small bug in linux keyboard destructor
  2. Updated "TEST" demos to work on SDL
  3. allow SDL to be specified on any platform as a compile directive
  4. added CMAKE config scripts as an alternative to autoconf (currently only cmake support enabling SDL on linux)
  5. rework and update the SDL platform code to work with the current codebase
  6. implement Joystick support for the SDL backend
